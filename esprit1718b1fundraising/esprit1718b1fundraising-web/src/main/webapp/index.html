Introduction

Studies is a very flexible template, you can easy use it for a university, as a tutorial page or as a job portal. The style is really clean and minimalistic. The theme is 100% customizable.

Support

info@nicolaskayser.com

How can I add styles on the sections?

You have two styleing options for the sections
1
Add the class bg-grey for an grey background, or b-b for an grey border at the bottom of the section.

How can I edit the submenu?

You have to switch the submenu from hidden to visible.
1
Select the submenu
2
Set the display none to block

How can I edit the home slideshow?

You must edit the slideshow classes, because it´s an background slideshow.
1
Select slide-1 of the hero-slideshow
2
Scroll to the background image and change it
3
If you like to add a new slide, add a new slide from the slideshow settings and get the slide the class slide-x (for example: the next slide will be slide-3)
4
Edit the background image of the new slide (important: you must set the background position to cover)

How can I edit the Clients labels?

You have to switch the labels from hidden to visible.
1
Select the logo-slideshow-tooltip
2
Set the opacity 0 to 100
3
Now you can edit the labels, if you are ready set the opacity again from 100 to 0
